What's New In KLatexFormula 3.2

This screen can be opened at all times from the "About" window accessible with the button in expanded mode.

Now a bigger preview can be opened on request after evaluating a formula (Shortcut: F2 key).

A new export mime-type profiles system allows you to specify precisely which formats you would like to export when using copy/paste and drag/drop. The formats are organized in profiles which you can select in a menu below the DRAG/COPY buttons.

You can now re-open saved PNG images (try dragging them onto the editor).

LaTeX' Computer Modern Sans Serif font is used as the default application font. Don't like it? Choose your preferred application font.

KLatexFormula now by default instructs ghostscript to outline fonts in EPS/PDF files. This fixes exports to eg. Adobe Illustrator.

The internals of the library were completely redesigned for greater flexibility. Read more on this subject on the wiki.

A small plugin for KDE 4's KTextEditor/KatePart was written to integrate KLatexFormula to any application that uses KTextEditor (eg. Kate, Kile)
