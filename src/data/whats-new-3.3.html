What's New In KLatexFormula 3.3

This screen can be opened at all times from the "About" window accessible with the button in expanded mode.

Mind that this version of KLatexFormula is a Beta release! Please don't hesitate to report your problems or missing features.

At last, KLatexFormula's window can be resized! Now your window can grow as large as you need it to display your large equations.

A revised workflow now produces better output quality and is more robust and flexible.

The new workflow allows custom user scripts, such as for compiling Feynman diagrams. See the Archive tab of the library for an example.

KLatexFormula has now reduced idle CPU consumption. Also by default formulas are not compiled in real time when on laptop battery power.
